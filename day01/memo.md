- 그래프 표현
- BFS / DFS

# 용어 정리
- 알고리즘
: 문제를 푸는 순서
- 자료구조
: 데이터를 효율적으로 담는 그릇
- 스택 (Stack)
: 마지막에 쌓은 것을 먼저 꺼내는 구조 (LIFO)
- 큐 (Queue)
: 버스 정류장 줄처럼 먼저 온 사람이 먼저 타는 구조 (FIFO)
- 그래프 (Graph)
: 지하철 노선도처럼 노드(점)과 간선(선)으로 연결 관계
- 노드 (Node)
: 지하철 노선도에서의 '역'
- 간선 (Edge)
: 역과 역을 잇는 '철길' / 노드와 노드 사이를 연결하는 선
- BFS 
: 내 옆에 있는 가까운 노드들부터 먼저 다 확인하고, 그 다음 단계로 넘어가는 방식
- DFS
: 갈 수 있는 끝까지 일단 들어갔다가 막히면 다시 돌아오는 방식
- 재귀 (Recursion)
: 함수가 자기 자신을 다시 호출하는 것
- 방문 (visited)
: 탐색할 때 "나 여기 이미 와봤어!"라고 체크 표시를 하는 것
- LIFO(Last-In, First-Out): 마지막에 들어온 게 먼저 나감 (스택 - 프링글스 통 과자)
- FIFO(First-In, First-Out): 먼저 들어온 게 먼저 나감 (큐 - 터널을 통과하는 자동차)
